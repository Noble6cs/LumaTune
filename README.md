# LumaTune
基于洛雪音乐LX Music，美化了流光播放器和歌单页面。

## 流光律首次公开版本

本项目是基于 [LX Music（洛雪音乐助手）](https://github.com/lyswhut/lx-music-desktop) 开源项目修改的非官方版本，
不代表 LX Music 官方立场。

播放器背景算法参考 [jayfunc/BetterLyrics](https://github.com/jayfunc/BetterLyrics)。

### 主要修改

- 我的列表新增所有列表封面栏目
- 新增动态歌单与收藏封面效果
- 优化播放器背景显示
- 支持前台状态下的播放快捷键

请遵守原项目许可证与补充协议，支持正版音乐。
